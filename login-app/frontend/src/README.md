# React + Node Login Application

This is a simple full-stack login application built with React and Node.js.

## Features

- Login page with username and password
- Backend API validation
- Redirect to welcome page after successful login
- Error message for invalid credentials
- Remember username using localStorage

## Tech Stack

Frontend:
- React
- React Hooks
- Axios
- React Router

Backend:
- Node.js
- Express
- CORS
